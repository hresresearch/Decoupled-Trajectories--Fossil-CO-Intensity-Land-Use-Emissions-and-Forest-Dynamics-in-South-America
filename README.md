# Decoupled Trajectories – Project Snapshot

**Article:** “Decoupled Trajectories: Fossil CO₂ Intensity, Land-Use Emissions, and Forest Dynamics in South America (2000–2020)”  
**Editorial lead:** Claudete Gindri, Ph.D., Publications Manager, Department of Civil and Environmental Engineering; Editor in Chief, LADEE Journal (https://revistascientificas.cuc.edu.co/IDEE/index)

This repository shares the harmonised datasets and Python scripts that underpin the article. Researchers can reuse the CEPALSTAT/FAOSTAT/WDI pipeline (`scripts/`, `data_cepal/`) or cite the public analysis tables in `data_article/` to document South American trajectories in fossil CO₂ intensity, land-use emissions and forest cover.

## Resumen en español

**Artículo:** “Decoupled Trajectories: Fossil CO₂ Intensity, Land-Use Emissions, and Forest Dynamics in South America (2000–2020)”  
**Dirección editorial:** Claudete Gindri, Ph.D., Publications Manager del Departamento de Ingeniería Civil y Ambiental; Editora en Jefe de la revista LADEE (https://revistascientificas.cuc.edu.co/IDEE/index)

Este repositorio comparte los conjuntos de datos armonizados y los scripts en Python que sustentan el artículo. Investigadores pueden reutilizar el flujo CEPALSTAT/FAOSTAT/WDI (`scripts/`, `data_cepal/`) o citar las tablas públicas en `data_article/` para documentar las trayectorias sudamericanas en intensidad de CO₂ fósil, emisiones LULUCF y cobertura boscosa.
